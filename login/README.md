# Aplikasi Login Flutter

## Deskripsi
Aplikasi Flutter sederhana dengan fitur login, lupa password, dan dashboard admin.

## Fitur
- Login validation
- Forgot password
- Dashboard admin
- Logout
- ListView data
- Snackbar dan Dialog

## Cara Menjalankan
```bash
flutter pub get
flutter run
```
